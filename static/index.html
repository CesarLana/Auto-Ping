Sincronização com Microsoft Intune

Importe a lista de dispositivos diretamente do Microsoft Intune (Endpoint Manager) para atualizar a base de colaboradores de forma automática.

Como exportar o arquivo do Intune:

  1. Acesse o portal do Microsoft Endpoint Manager (Intune).
  2. Navegue em Dispositivos > Todos os dispositivos.
  3. Clique no botão Exportar no topo da tabela e faça o download do arquivo CSV.
